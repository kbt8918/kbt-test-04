## 프로젝트 목표

고령의 부모님 위치를 가족이 실시간으로 확인하고, 긴급 상황 시 신속하게 대응할 수 있도록 지원하는 반응형 웹 기반 가족 안전 서비스를 구축한다.
디지털 기기에 익숙하지 않은 시니어도 단순 조작만으로 긴급 알림을 발송할 수 있어야 하며, 가족 간 실시간 소통과 위치 모니터링을 하나의 서비스에서 제공한다.

## 핵심 기능

1. **실시간 위치 공유** — 부모님 GPS 위치를 가족에게 실시간 전송 (반응형 웹)
2. **긴급 SOS 알림** — 부모님이 버튼 한 번으로 모든 가족에게 위치+시각 포함 푸시 알림 발송
3. **가족 위치 지도** — 카카오맵 API 기반 실시간 위치 마커 + 이동 이력(7일) 조회
4. **가족 채팅 및 전화 연결** — 가족 그룹 채팅방 + 부모님에게 바로 전화 연결 버튼
5. **관리자 회원 관리 및 커뮤니케이션** — 회원(가족 그룹) 관리, SMS 및 카카오 알림톡 발송

## 대상 사용자

| 구분 | 설명 | 주요 니즈 |
|------|------|-----------|
| 부모님 | 디지털 기기 활용에 익숙하지 않은 시니어 | 쉬운 조작, 큰 버튼, 긴급 상황 알림 |
| 가족 구성원 | 스마트폰에 익숙한 자녀·배우자 | 부모님 위치 확인, 빠른 연락, 가족 소통 |
| 서비스 관리자 | 운영사 담당자 | 회원 관리, 이상 징후 모니터링, 고객 커뮤니케이션 |

## 비기능 요건

- 성능: 위치 업데이트 지연 5초 이내, 긴급 알림 전송 3초 이내
- 보안: 위치 데이터 HTTPS/TLS 암호화, 위치정보법 준수, 개인정보보호법 준수
- 접근성: 모바일(360px~) · 태블릿(768px~) · PC(1280px~) 반응형, 글자 최소 18px, 버튼 최소 48×48px, WCAG AA 대비 충족
- 가용성: Uptime 99.5% 이상
- 브라우저: Chrome · Safari · Samsung Internet 최신 2버전
