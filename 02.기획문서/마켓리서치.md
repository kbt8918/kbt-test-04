## 1. 리서치 목적

부모님 위치 확인 서비스의 **시장 진입 타당성을 검증**하고, 서비스기획서·요구사항정의서 작성의 근거 데이터를 확보하기 위해 아래 5가지 핵심 질문을 탐구한다.

| # | 핵심 질문 |
|---|-----------|
| Q1 | 한국 시니어 케어·위치추적 서비스 시장의 규모와 성장성은? |
| Q2 | 국내외 주요 경쟁 서비스의 기능·가격·차별점은? |
| Q3 | 시니어(부모님)와 보호자(자녀)가 기존 솔루션에서 느끼는 불만과 충족되지 않은 니즈는? |
| Q4 | 관련 기술 동향과 규제 환경은 어떻게 변화하고 있는가? |
| Q5 | 서비스의 차별화 포인트와 진입 전략은? |

### 2-1. 글로벌 가족 위치추적 앱 시장 전망 보고서

| 항목 | 내용 |
|------|------|
| 출처 | Dataintelo — Family Tracking App Market Research Report 2034 |
| 발표일 | 2025년 |
| 자료 유형 | 시장조사 보고서 |
| 키워드 | 가족위치추적, 글로벌시장, CAGR, 아시아태평양 |

**요약**

글로벌 가족 위치추적 앱 시장은 2025년 약 38억 달러(약 5.1조 원)에서 2034년 97억 달러(약 13조 원)로 성장할 것으로 전망된다. 연평균 성장률(CAGR) 10.9%이며, 아시아태평양 지역은 CAGR 13.4%로 전 세계 최고 성장세를 보인다. 스마트폰 보급 확대, 고령화, 안전 관련 불안 심리 증가가 주요 성장 동인이다.

| 시장 지표 | 수치 |
|-----------|------|
| 시장 규모 (2025) | 38억 달러 (약 5.1조 원) |
| 시장 규모 (2034, 전망) | 97억 달러 (약 13조 원) |
| 글로벌 CAGR | 10.9% |
| 아시아태평양 CAGR | 13.4% (전 세계 최고) |

**주요 인사이트**
- 아시아태평양 지역이 전 세계 최고 성장률(CAGR 13.4%) → 한국 시장 진입 최적 시점
- 시니어 케어·자녀 안전 동시 타겟 서비스가 가장 빠르게 성장
- 안드로이드 기반 앱이 시장 매출의 약 48.6%로 단일 플랫폼 최대 점유 (Dataintelo 공식 수치), 국내 안드로이드 보급률(약 70~80%)을 고려 시 안드로이드 우선 개발 + iOS 연동 전략 유효

**프로젝트 적용 방안**
글로벌 성장세가 확인된 시장에서 한국어·한국 법규 특화 서비스로 로컬 니즈를 충족하면 충분한 경쟁력 확보 가능. MVP 출시 후 글로벌 확장 로드맵 검토의 근거로 활용.

### 2-2. 한국 실버산업 시장 규모 및 성장 전망

| 항목 | 내용 |
|------|------|
| 출처 | 전자신문 / 한국경제 |
| 발표일 | 2022년 |
| 자료 유형 | 언론 보도 |
| 키워드 | 실버산업, 초고령사회, B2G, 복지서비스 |

**요약**

한국 실버산업(고령자 대상 재화·서비스 전체) 시장은 2020년 72조 원에서 2030년 168조 원으로 2배 이상 성장이 전망된다. 2025년 한국은 65세 이상 인구가 전체의 20%를 넘어 초고령사회에 공식 진입하였으며, 이는 돌봄·안전·커뮤니케이션 수요의 폭발적 증가를 의미한다.

| 시장 지표 | 수치 |
|-----------|------|
| 시장 규모 (2020) | 72조 원 |
| 시장 규모 (2030, 전망) | 168조 원 |
| 10년 성장률 | 133% |
| 65세 이상 인구 비율 | 20% 이상 (초고령사회 기준) |

**주요 인사이트**
- 실버산업은 의료·요양에서 IT·안전 서비스로 영역이 확대되는 추세
- 5명 중 1명이 65세 이상인 사회 구조로 시장 지속 확대 확실
- 정부·지자체의 복지 예산 증가로 B2G 시장도 동반 성장

**프로젝트 적용 방안**
시장 성장이 구조적으로 확정된 상황에서 조기 진입 시 브랜드 인지도 선점 가능. 지자체 복지 사업 연계 B2G 채널을 부가 수익 모델로 검토.

### 2-3. 치매 환자 실종 현황 및 배회감지기 보급률

| 항목 | 내용 |
|------|------|
| 출처 | 보건복지부 / 디멘시아뉴스 / 푸드투데이 |
| 발표일 | 2024년 |
| 자료 유형 | 정책 통계 |
| 키워드 | 치매, 실종, 배회감지기, 보급률 |

**요약**

2024년 기준 국내 치매 환자 수는 910,898명으로 2020년(529,475명) 대비 72% 급증했다. 같은 해 치매 환자 실종신고는 15,502건으로, 이는 「실종아동등의 보호 및 지원에 관한 법률」상 법정 보호 대상 취약군(아동·치매환자·장애인 합계 약 4.9만 건) 중 약 31.2%를 차지한다. 그러나 이를 예방할 수 있는 배회감지기 보급률은 단 6.1%에 불과하며, 미보급 원인의 47.9%가 '정보 부족'이다.

| 시장 지표 | 수치 |
|-----------|------|
| 치매 환자 수 (2024) | 910,898명 |
| 연간 실종신고 건수 | 15,502건 |
| 배회감지기 보급률 | 6.1% |
| 미보급 원인 1위 | 정보 부족 (47.9%) |
| 수색 시간 단축 효과 | 평균 660분 → 55분 |

**주요 인사이트**
- 배회감지기 보급률 6.1% = 시장 침투 가능성 93.9%가 아직 미개척
- '정보 부족'이 최대 장벽 → 서비스 인지도 확보가 최우선 과제
- 수색 시간 단축 효과가 검증됨 (약 92% 단축)

**프로젝트 적용 방안**
타겟 사용자 중 치매 초기·경도인지장애 시니어를 포함하도록 UX 설계. 복지관·요양원 등 오프라인 채널을 통한 홍보 전략 필수. 정부 무상 보급 사업과 연계 가능성 검토.

### 2-4. iSharing — 글로벌 시니어 특화 위치공유 앱

| 항목 | 내용 |
|------|------|
| 출처 | iSharing 공식 웹사이트 / 서비스 리뷰 |
| 발표일 | 2024년 |
| 자료 유형 | 경쟁사 분석 |
| 키워드 | iSharing, 비활동감지, 안심존, 구독모델 |

**요약**

iSharing은 국내에서 가장 많이 사용되는 가족 위치공유 앱 중 하나로, 365일 이동기록 보관, 비활동 감지 알림, 안심존 이탈 알림, SOS 긴급 알림을 제공한다. 무료 플랜과 유료 플랜(월 8,900원 / 연 74,900원)을 운영하며 시니어 케어 특화 기능으로 국내 사용자 기반을 확보하고 있다.

| 비교 항목 | 내용 |
|-----------|------|
| 이동 이력 | 365일 |
| 요금 | 무료 / 월 8,900원 / 연 74,900원 |
| 차별 기능 | 비활동 감지 (설정 시간 동안 앱 조작·이동이 없을 때 보호자에게 푸시 알림 발송) |
| 약점 | 스마트폰 필요, 가족 채팅·전화 통합 없음 |

**주요 인사이트**
- '비활동 감지' 기능이 경쟁 앱 대비 고유한 차별점 — 단, 하드웨어 낙상 센서가 아닌 소프트웨어 기반 모션 감지(스마트폰 무조작·무이동 감지) 방식임을 유의
- 무료 플랜으로 진입 장벽 낮춤, 유료 전환율로 수익 창출하는 Freemium 모델 검증
- 스마트폰이 있어야 작동 — 디지털 취약 시니어에게는 여전히 한계

**프로젝트 적용 방안**
iSharing 대비 차별점으로 (1) 시니어 전용 초간편 UI, (2) 가족 채팅·전화 통합, (3) 관리자 어드민 기능을 강조. Freemium 모델을 참고하여 수익 모델 수립.

| 비교 항목 | 내용 |

- 가족 채팅 기능이 포함되어 있어 위치+소통 통합 수요 검증됨

### 2-6. Gper(스파코사/원더온) — 국내 LoRa 기반 위치추적기

| 항목 | 내용 |
|------|------|
| 출처 | 스파코사 공식 웹사이트 |
| 발표일 | 2024년 |
| 자료 유형 | 경쟁사 분석 |
| 키워드 | Gper, LoRa, 하드웨어, 저전력 |

**요약**

Gper는 SK텔레콤의 LoRa(저전력 장거리 통신망) 인프라를 활용하는 초소형 GPS 단말기 서비스로, 연동 앱 FAMY는 무료다. 스마트폰 없이 작동하므로 디지털 취약 시니어에게 적합하고, 하루 이상 배터리 지속이 가능하다.

| 비교 항목 | 내용 |
|-----------|------|
| 요금 모델 | 단말기 구매 + FAMY 앱 무료 |
| 기반 기술 | SKT LoRa 네트워크 |
| 강점 | 배터리 지속력, 스마트폰 불필요, 착용 편의 |
| 약점 | 하드웨어 구매 비용, 가족 채팅·전화 없음, UI 단순 |

**주요 인사이트**
- 스마트폰 불필요 = 디지털 취약 시니어의 진입 장벽 최소화
- LoRa 기반 초저전력 통신으로 배터리 소모 우려 해소
- 하드웨어 단말기 의존으로 가족 소통 기능(채팅·전화) 완전 부재

**프로젝트 적용 방안**
하드웨어 없이 스마트폰 앱만으로 동일한 위치 공유를 제공하는 소프트웨어 접근으로 초기 비용 절감. 단, 스마트폰 미사용 시니어 대상으로 향후 하드웨어 파트너십 검토.

### 2-7. 보건복지부 배회감지기 보급 사업

| 항목 | 내용 |
|------|------|
| 출처 | 보건복지부 / 국민건강보험공단 |
| 발표일 | 2024년 |
| 자료 유형 | 정부 정책 |
| 키워드 | 배회감지기, 보건복지부, B2G, 치매안심센터 |

**요약**

보건복지부는 치매안심센터를 통해 치매 환자 대상 배회감지기를 무상·저가 임대 형태로 보급한다. GPS·LoRa 기반 목걸이형·손목형·신발형 단말기를 지원하며, 경찰청·소방청과 연계하여 실종 발생 시 수색 지원 체계를 운영한다.

| 항목 | 내용 |
|------|------|
| 치매안심센터 수 | 전국 256개소 |
| 보급 형태 | 무상/저가 임대 |
| 연계 기관 | 경찰청, 소방청, 건강보험공단 |
| 보급 형태 | 목걸이형, 손목형, 신발형(꼬까신) |

**주요 인사이트**
- 공공 무상 보급 채널 = B2G(정부·지자체) 파트너십 기회
- 착용 거부가 핵심 문제 → 스마트폰 앱 기반 서비스는 이 문제 우회 가능
- 치매안심센터(전국 256개) = 오프라인 홍보·배포 거점으로 활용 가능

**프로젝트 적용 방안**
단기는 민간 앱 서비스로 출시, 중기는 지자체·치매안심센터 연계 B2G 계약으로 안정적 사용자 기반 확보. 정부 연계 시 개인정보보호법·위치정보법 이중 준수 체계 선제 구축 필요.

| 키워드 | 에이지테크, AgeTech, AI돌봄, 디지털취약계층 |

**요약**

정부는 '디지털 대전환 플래그십 프로젝트'(약 3,000억 원 규모)를 추진 중이며, IoT·AI·GPS를 결합한 스마트 돌봄 서비스가 핵심 분야다. 에이지테크는 (1) AIP Tech(자립생활 지원), (2) CareTech(돌봄 로봇·AI 모니터링), (3) 기술 수용 서비스(디지털 취약계층 접근성)의 3축으로 발전 중이다.

| 항목 | 내용 |
|------|------|
| 정부 투자 규모 | 약 3,000억 원 (디지털 대전환 플래그십) |
| 핵심 기술 | IoT + AI + GPS + 클라우드 |
| 발전 방향 | 위치 공유 → AI 이상행동 감지 → 낙상 예측 |

**주요 인사이트**
- 단순 위치 공유를 넘어 AI 기반 이상행동 패턴 감지·낙상 예측으로 진화하는 추세
- 디지털 취약계층(시니어) 접근성 제고가 정책 과제 → 시니어 친화 UX가 경쟁력
- 정부 지원 사업과의 연계 가능성이 높은 분야

**프로젝트 적용 방안**
MVP에는 GPS+실시간 위치 공유로 출발하되, Phase 2·3에서 비활동 감지·이상행동 분석 AI 기능 로드맵 수립(PRD 2차 출시 항목). 정부 지원 사업 공모 대상으로 기술 문서 정비.

### 2-9. 서울시×SKT '스마트 지킴이' 공공-민간 협력 사례

| 항목 | 내용 |
|------|------|
| 출처 | 정책브리핑 / 더인디고 |
| 발표일 | 2023년 |
| 자료 유형 | 케이스 스터디 |
| 키워드 | 스마트지킴이, 서울시, SKT, 공공민간협력 |

**요약**

서울시와 SK텔레콤이 협력하여 GPS 기반 치매노인 위치추적 단말기 '스마트 지킴이'를 무상 보급했다. 양천구 시범사업에서 7건의 실종 사건을 대부분 1시간 이내 해결하는 성과를 거뒀다. 기존 평균 수색 시간 660분 대비 55분으로 약 92% 단축되어 서비스 효과가 검증됐다.

| 항목 | 내용 |
|------|------|
| 시범 지역 | 서울시 양천구 |
| 실종 해결 건수 | 7건 |
| 수색 시간 | 660분 → 55분 (92% 단축) |
| 협력 구조 | 지자체 + 통신사 + 서비스 제공자 |

**주요 인사이트**
- 공공-민간 협력 모델의 성과 검증 완료 — B2G 전략의 실행 근거
- 실종 수색 시간 92% 단축 = 명확한 사회적 임팩트, 마케팅 근거로 활용 가능
- 지자체 예산 집행(무상 보급) + 민간 앱 연계가 성공 공식

**프로젝트 적용 방안**
서비스 출시 후 지자체 시범사업 제안서 작성 시 수색 시간 단축 효과를 핵심 성과지표(KPI)로 활용. API 개방으로 지자체 연계 시스템과 통합 가능한 구조로 설계.

### 2-10. 시니어 디지털 기기 활용 현황

| 항목 | 내용 |
|------|------|
| 출처 | 와이즈앱 리테일굿즈 |
| 발표일 | 2024년 |
| 자료 유형 | 통계 보고서 |
| 키워드 | 액티브시니어, 시니어UX, 스마트폰사용률, 카카오톡 |

**요약**

2024년 국내 60세 이상 시니어의 스마트폰 사용률은 꾸준히 증가하고 있으며, '액티브 시니어(Active Senior)'로 불리는 디지털 친화적 고령자 집단이 부상하고 있다. 카카오톡·유튜브 사용률은 높으나, 새로운 앱 설치·복잡한 UI 조작에는 여전히 저항감이 크다.

| 항목 | 내용 |
|------|------|
| 60세 이상 스마트폰 사용률 | 꾸준히 증가 추세 |
| 고사용 앱 | 카카오톡, 유튜브 |
| 선호 UI | 최소 버튼, 큰 글씨(18px+), 직관적 아이콘 |
| 저항 요소 | 새 앱 설치, 복잡한 UI 조작 |

**주요 인사이트**
- 카카오톡 사용률이 높음 → 카카오 알림톡 기반 알림이 시니어에게 가장 친숙
- 새 앱 설치 저항감 → 가족(자녀)이 부모님 기기에 설치해주는 '대리 온보딩' 방식이 현실적
- 글씨 18px+, 버튼 48×48px+ → PRD 비기능 요구사항의 근거 데이터로 직접 활용 가능

**프로젝트 적용 방안**
부모님 화면(Parent View)을 SOS 버튼 중심의 극단적 단순 UI로 설계. 초기 설정은 자녀가 대신 진행하는 '온보딩 대리 설치' 흐름 포함. 카카오 알림톡을 관리자 알림의 주채널로 확정 (착수보고서 기술 스택 반영).

### 2-11. 위치정보 보호법 및 개인정보보호법 규제 환경

| 항목 | 내용 |
|------|------|
| 출처 | 방송통신위원회 / 개인정보보호위원회 |
| 발표일 | 2023~2024년 |
| 자료 유형 | 법규 분석 |
| 키워드 | 위치정보법, 개인정보보호법, 방통위신고, 동의절차 |

**요약**

한국의 위치정보 서비스는 '위치정보의 보호 및 이용 등에 관한 법률(위치정보법)'과 '개인정보보호법'의 이중 규제를 받는다. 본 서비스는 스마트폰 GPS를 활용해 서비스를 제공하는 **위치기반서비스사업자**에 해당하며, 방통위에 신고(심사 없이 즉시 처리)해야 한다. 타인의 위치를 추적·수집할 경우 정보주체(부모님)의 명시적 동의가 필수이며, 위반 시 형사 처벌을 받는다.

> **위치정보법 사업자 구분**: 위치정보사업자(등록제, GPS 등을 직접 수집·제공)와 위치기반서비스사업자(신고제, 수집된 위치정보를 이용해 서비스 제공)는 별개이며, 본 서비스는 후자에 해당한다.

| 규제 항목 | 내용 |
|-----------|------|
| 위반 시 제재 | 3년 이하 징역 또는 3천만 원 이하 벌금 (위치정보법 제40조) |
| 필수 요건 | 위치기반서비스사업자 방통위 신고 (신고제 — 심사 없이 즉시 처리) |
| 동의 요건 | 정보주체(부모님) 명시적 사전 동의 (위치정보법 제15조) |
| 관리자 접근 | 개인 위치 데이터 접근 금지 (착수보고서 Out of Scope) |

**주요 인사이트**
- 위치기반서비스사업자 미신고 운영도 형사처벌 대상 → 서비스 출시 전 방통위 신고 필수 (착수보고서 R-01)
- 부모님(정보주체)의 명시적 동의 없이 자녀가 위치를 추적하는 것은 불법 (부부·부모자녀 관계도 예외 없음)
- 자녀 대리 설치 시 부모님 단말기 화면에서 독립적인 명시적 사전 동의 프로세스 구현 필수

**프로젝트 적용 방안**
서비스 출시 전 위치기반서비스사업자 방통위 신고 진행. 부모님 회원가입 시 위치정보 수집·이용 동의를 별도 화면으로 분리하여 명시적 동의 확보. 개인정보처리방침 페이지 필수 포함.
