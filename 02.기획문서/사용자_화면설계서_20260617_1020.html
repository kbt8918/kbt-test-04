BE 연동: 없음.

회원가입 탭		 로그인·회원가입 모드 전환 탭. 모드에 따라 하단 폼 노출 토글.

노출조건: 항상 표시.
입력조건: 탭 클릭으로 모드 전환. 기본값: 로그인.
출력조건: 선택 탭 active 스타일. 전환 시 폼 초기화.
예외 Case: 탭 전환 시 모든 입력값 초기화(클리어).

노출조건: 로그인·회원가입 탭 모두 표시.
입력조건: 숫자 11자리, 010 시작. 자동 하이픈(-) 삽입. 숫자 외 입력 차단.
출력조건: 완료: 010-XXXX-XXXX 포맷. 미입력: placeholder “010-1234-5678”.
예외 Case: 010 외 번호 입력 시 경고 토스트. 11자리 미만 시 버튼 비활성.

입력조건: 단일 선택 필수. 기본값: 미선택.
출력조건: 선택 역할 active 카드 스타일. 미선택 시 시작 버튼 비활성.
예외 Case: 미선택 상태에서 시작 클릭 시 “역할을 선택해주세요” 토스트.
BE 연동: 선택값(parent/family) 회원가입 API 페이로드에 포함.
5CTA 영역약관 동의 + 시작하기 [필수] 개인정보처리방침 동의 체크 시 CTA 활성화. 미체크 시 비활성.

노출조건: 항상 표시.
입력조건: 체크박스 클릭(동의/철회). 미동의 시 버튼 disabled.
출력조건: 동의 → primary 색상 버튼 활성. 미동의 → gray 비활성.
예외 Case: 미동의 상태 버튼 클릭 시 “약관 동의가 필요합니다” 토스트.
BE 연동: POST /api/auth/login 또는 /api/auth/register (인증 플로우 시작).
6OAuth 영역Google로 계속하기 구글 OAuth 원클릭 로그인/가입. 하단 데모 둘러보기 텍스트 버튼.

노출조건: 항상 표시.
입력조건: 버튼 탭 → Google OAuth 플로우 시작.
출력조건: OAuth 성공 시 역할 지정(신규) 또는 홈 이동(기존). 실패 시 토스트.
예외 Case: 취소/네트워크 오류 시 “로그인 실패” 토스트. 재시도 가능.
BE 연동: POST /api/auth/google { id_token } → JWT 발급. 신규 사용자 /api/auth/register?provider=google 연계.

Description

No.영역요소설명
1진행 영역단계 진행 표시 현재 단계(N/4)와 진행 바.

노출조건: 항상 표시.
출력조건: 단계에 맞게 바 길이·숫자 업데이트.
인터랙션: 단계 이동 시 진행 바 너비 증가 애니메이션(ease-in-out 0.3s).
예외 Case: 단계 수 로드 실패 시 진행 바 숨김.
2콘텐츠 영역안내 문구 해당 단계의 핵심 안내 텍스트.

노출조건: 단계 전환 시 해당 텍스트로 교체.
출력조건: 단계에 따라 문구 변경.
인터랙션: 단계 전환 시 텍스트 페이드-인(0.2s).
예외 Case: 텍스트 누락 시 기본 안내 문구 폴백.
3폼 영역입력/동의 영역 초대 코드(6자리) 또는 위치정보 동의 체크 영역.

노출조건: 단계별 조건에 따라 코드 입력 필드 또는 동의 체크박스로 전환.
입력조건: 코드 입력: 숫자 6자리. 동의: 체크박스 클릭.
출력조건: 코드 입력 시 실시간 유효성 표시.
인터랙션: 코드 각 자리 입력 시 시각적 피드백. 동의 체크 시 체크마크 애니메이션.
예외 Case: 코드 불일치 시 인라인 에러. 3회 실패 시 30초 잠금.
BE 연동: POST /api/group/join. PUT /api/user/location-consent.
4CTA 영역다음/시작 버튼 조건 충족 시 활성화. 위치 동의 단계에서는 Geolocation 권한 트리거.

노출조건: 항상 표시 (조건 미충족 시 비활성).
입력조건: 버튼 탭 → 다음 단계 이동 또는 권한 요청 팝업.
출력조건: 활성: primary 색상. 비활성: gray.
인터랙션: 활성화 탭 시 ripple 효과. 권한 요청 시 시스템 팝업 호출.
예외 Case: 권한 거부 시 위치 권한 안내 모달.
BE 연동: 마지막 단계 완료 시 POST /api/user/onboarding-complete.

Description

No.영역요소설명
1상태 영역위치 공유 상태 카드 현재 공유 on/off를 큰 텍스트·펄스 애니메이션으로 표시.

노출조건: 항상 표시.
출력조건: ON: 녹색 펄스 + 위치 전송 중. OFF: 회색 + 위치 공유 꺼짐.
인터랙션: ON 상태 시 연속 펄스 애니메이션(0.8s 루프).
예외 Case: 위치 권한 없을 시 권한 필요 경고 카드로 대체.
BE 연동: GET /api/user/location-status. WebSocket /ws/location.
2CTA 영역가족과 대화하기 음성 채팅(SCR-014)으로 진입하는 시니어용 대형 버튼.

노출조건: 항상 표시.
입력조건: 버튼 탭 → SCR-014 음성 채팅 화면으로 이동.
인터랙션: 탭 시 ripple 효과 + SCR-014 슬라이드-인 전환.
예외 Case: 오프라인 시 인터넷 연결 확인 토스트, 화면 전환 차단.
3SOS 영역SOS 긴급 호출 버튼 화면 중앙 대형 레드 원형 버튼. 1회 터치 시 2초 카운트다운 오버레이.

노출조건: 항상 표시.
입력조건: 1회 터치 → 2초 카운트다운. 카운트다운 중 재터치 → 취소.
출력조건: 카운트다운 완료 시 모든 가족에게 FCM 푸시 + GPS 좌표 전송.
인터랙션: 탭 시 햅틱(진동) 피드백. 카운트다운 원형 오버레이 표시. 취소 시 원상 복귀 애니메이션.
예외 Case: GPS 없을 시 최근 저장 좌표로 대체 전송. 알림 실패 시 SMS 폴백.
BE 연동: POST /api/sos/trigger { lat, lng, userId }.
4설정 영역위치 공유 토글 실시간 GPS 수집/전송 일시 중단·재개.

노출조건: 항상 표시.
입력조건: 토글 스위치 클릭.
출력조건: ON: 녹색 + GPS 재개. OFF: 회색 + GPS 중단.
인터랙션: 슬라이드 토글 애니메이션(0.2s). 동의 철회 상태에서 탭 시 확인 모달 fade-in.
예외 Case: 동의 철회 상태에서 ON 시도 시 위치 동의 안내 모달.
BE 연동: PUT /api/user/location-sharing { active: bool }.
5헤더 영역설정 바로가기 우상단 톱니 아이콘. 부모님 설정(SCR-004)으로 이동.

노출조건: 항상 표시.
입력조건: 아이콘 탭 → SCR-004 화면으로 이동.
인터랙션: 탭 시 아이콘 active(눌림) 상태. 설정 화면 슬라이드-인.

Description

No.영역요소설명

노출조건: 항상 표시.

출력조건: 약관 문서 렌더링.
인터랙션: 탭 시 리스트 아이템 active 배경(0.1s).
예외 Case: 오프라인 시 오프라인 토스트.
3위험 영역위치 공유 동의 철회 Danger 버튼. 탭 시 확인 모달 → 철회 시 GPS 수집 중단.

노출조건: 항상 표시.
입력조건: 버튼 탭 → 2단계 확인 모달. 확인 탭 → 철회 처리.
출력조건: 철회 완료 시 위치 공유 토글 비활성화.
인터랙션: Danger 버튼 탭 시 햅틱 피드백. 확인 모달 fade-in. 철회 처리 중 로딩.
예외 Case: 서버 오류 시 철회 처리 실패 토스트.
BE 연동: DELETE /api/user/location-consent.

인터랙션: 입력 완료 시 자동 유효성 검사(실시간). QR 스캔 시 카메라 오픈.
예외 Case: 코드 불일치 시 인라인 에러. 만료 코드 시 별도 안내.

1헤더 영역헤더 음성 채팅 헤더. 좌상단 뒤로가기 → SCR-008 복귀.

노출조건: 항상 표시.
입력조건: 뒤로가기 탭 → SCR-008 복귀.
인터랙션: 슬라이드-아웃 전환.
2채팅 영역채팅 메시지 윈도우 큰 폰트 말풍선 타임라인. 시니어 가독성 최적화.

노출조건: 항상 표시.
출력조건: 큰 폰트 말풍선. 새 메시지 수신 시 자동 스크롤.
인터랙션: 새 메시지 수신 시 부드럽게 스크롤-다운. 무한 스크롤(상단 로딩).
예외 Case: 로드 실패 시 불러오기 실패 안내.
BE 연동: GET /api/chat/messages. WebSocket /ws/chat.
3미리보기 영역인식 내용 미리보기 음성 인식 결과 실시간 표시. x 버튼으로 초기화.

노출조건: 음성 인식 활성 또는 텍스트 존재 시.
입력조건: x 버튼 탭 → 인식 텍스트 초기화.
출력조건: 실시간 인식 텍스트 표시.
인터랙션: 텍스트 입력 시 타이핑 효과. x 탭 시 텍스트 fade-out.
4입력 영역음성 입력 버튼 시니어용 대형 마이크 버튼. 탭 시 음성 인식 시작/중지.

노출조건: 항상 표시.
입력조건: 탭 → 인식 시작. 재탭 → 중지.
출력조건: 인식 중 펄스 애니메이션 표시.
인터랙션: 탭 시 햅틱 피드백. 인식 중 실시간 파형/펄스 표시. 완료 시 미리보기 텍스트 fade-in.
예외 Case: 마이크 권한 없을 시 권한 요청 모달. iOS Safari 미지원 시 키보드 안내.
BE 연동: Web Speech API. 완료 후 WebSocket emit chat:send.
5CTA 영역보내기·글자판 전환 보내기(음성→전송) 버튼 + 글자판 전환 버튼.

노출조건: 항상 표시.
입력조건: 보내기 탭 → 텍스트 전송. 글자판 탭 → 키보드 슬라이드-업.
출력조건: 전송 성공 시 메시지 추가 + 미리보기 초기화.
인터랙션: 보내기 탭 시 메시지 슬라이드-인. 글자판 전환 시 키보드 슬라이드-업.
예외 Case: 인식 텍스트 없을 시 보내기 비활성.
BE 연동: WebSocket emit chat:send.
