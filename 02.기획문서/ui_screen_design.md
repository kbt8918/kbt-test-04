#### 🔄 예외 처리 및 상태
* **동의 미체크/잘못된 번호 상태:** 조건 미충족 시 시작하기 버튼 비활성화 및 인라인/토스트 에러 제공.
* **미가입 번호 로그인:** 등록되지 않은 번호로 로그인 시도 시 "등록되지 않은 번호입니다. 회원가입을 진행해주세요." 인라인 에러.
* **중복 가입:** 이미 가입된 번호로 회원가입 시도 시 "이미 가입된 휴대폰 번호입니다. 로그인을 진행해주세요." 인라인 에러.
* **구글 계정 중복:** 이미 가입된 구글 계정으로 가입 시도 시 "이미 연동된 구글 계정입니다. 로그인 탭을 이용해주세요." 에러 배너 상단 표시.
* **로그인 랜딩 라우팅 (Smart Landing):**
  - 일반 회원가입(부모님): **온보딩** 화면으로 이동 (단, 초대 코드 보유 시 자동 참여 후 분기).
  - 일반 회원가입(가족 자녀): **가족 설정 화면**으로 이동.
  - 구글 신규 가입: 회원가입 폼을 생략하고 바로 **가족 설정 화면**으로 랜딩.
  - 로그인 후: 그룹 내 부모님이 한 분이라도 계시면 **가족 지도(SCR-006)** 로, 안 계시면 **가족 설정 화면**으로 지능형 분기 처리.
  - 관리자(`role=admin`): **관리자 화면(SCR-011)** 으로 이동.
* **초대 코드 자동 참여:** `/join?code=XXX` 진입 후 로그인·가입에 성공하면 보류된 초대 코드로 그룹에 자동 참여 시도하고 결과를 토스트로 안내.

### SCR-002. 온보딩 플로우 (1~4단계)

| 항목 | 내용 |
|------|------|
| URL | `/onboarding/step1` ~ `/onboarding/step4` |
| 플랫폼 | 모바일 웹 |
| 접근 권한 | 로그인 완료 + `role=parent` + 온보딩 미완료 사용자 (자녀 대리 설정 시나리오) |
| 관련 기능 | F-005 (온보딩), F-001 (번호 가입), F-004 (그룹 참여), F-006 (위치 동의) |
| 관련 API | `POST /api/groups/join`, `POST /api/location/consent` |

#### 📐 레이아웃 구조

```text
[온보딩 2단계: 그룹 참여]              [온보딩 3단계: 위치정보 동의]
+---------------------------------------+ +---------------------------------------+
|  안심맵 온보딩               [ 2 / 4 ] | |  안심맵 온보딩               [ 3 / 4 ] |
+---------------------------------------+ +---------------------------------------+
|                                       | |                                       |
|  자녀 휴대폰에 표시된                 | |  ⚠️ 부모님 직접 조작 필수              |
|  가족 초대 코드를 입력해 주세요.       | |  자녀의 대리 동의는 법적 처벌         |
|                                       | |  대상이 될 수 있습니다.               |
|  초대 코드 (6자리)                     | |                                       |
|  +---------------------------+         | |  [x] [필수] 개인 위치정보 수집·이용   |
|  |  A B C D E F              |         | |      및 제3자 제공 동의 [내용보기]    |
|  +---------------------------+         | |                                       |
|                                       | |  [ ] [선택] 혜택 및 알림 수신 동의    |
|  +---------------------------+         | |                                       |
|  |        다음 단계로        |         | |  +---------------------------+         |
|  +---------------------------+         | |  |        위치 공유 시작하기  |         |
|                                       | |  +---------------------------+         |
+---------------------------------------+ +---------------------------------------+
```

> 온보딩은 4단계로 구성됩니다. **1단계** 환영 인사("안녕하세요, 부모님!"), **2단계** 초대 코드 입력, **3단계** 위치정보 동의, **4단계** 완료("설정이 완료되었어요!"). 상단에는 단계 진행 표시(`N / 4`)와 진행 바가 항상 노출됩니다.

| **안심맵 시작하기 버튼** | Button | 4단계 완료 후 부모님 메인으로 진입 | `onboarded` 상태 저장 및 `localStorage` 진행값 제거 후 부모님 메인(SCR-003)으로 이동 |

#### 🔄 예외 처리 및 상태
