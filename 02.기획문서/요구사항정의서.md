## 1. 기능 요구사항 (Functional Requirements)

### 1.1 인증 및 회원 관리

| ID | 구분 | 요구사항 | 우선순위 | 인수 기준 |
|----|------|----------|----------|-----------|
| REQ-001 | 회원가입 | 휴대폰 번호 또는 구글 소셜 연동을 통해 회원가입이 가능해야 한다 | Must | 번호 입력 또는 구글 계정 원클릭으로 가입 완료, 구글 계정 중복 시 에러 배너 제공 |
| REQ-002 | 로그인 | 등록된 휴대폰 번호 또는 구글 계정으로 로그인할 수 있어야 한다 | Must | 인증 후 세션 발급, 자동 로그인 유지 (7일) |
| REQ-003 | 스마트 랜딩 라우팅 | 로그인/가입 완료 후 역할 및 그룹 상태에 따라 랜딩 화면이 지능적으로 분기되어야 한다 | Must | 부모님: 메인/온보딩 / 자녀: 회원가입 직후 가족설정, 로그인 시 그룹 내 부모님 존재 시 안심맵, 부재 시 가족설정 화면으로 자동 라우팅 |
| REQ-004 | 가족 그룹 생성 | 가족 구성원이 가족 그룹을 생성할 수 있어야 한다 | Must | 그룹 생성 시 6자리 초대 코드 발급, 데모/미가입 유저의 코드 복사는 차단됨 |
| REQ-005 | 가족 그룹 참여 | 초대 코드 직접 입력 또는 초대 링크 기반 자동 참여가 가능해야 한다 | Must | 코드 입력 또는 초대 링크(`/join?code=XXX`) 접속 시 백그라운드 자동 참여 완료, 최대 구성원 5명 (Free 플랜 기준) |

|----|------|----------|----------|-----------|

| REQ-008 | 동의 철회 | 부모님이 언제든 위치정보 제공 동의를 철회할 수 있어야 한다 | Must | 설정 화면에서 동의 철회 시 즉시 위치 수집 중단 및 가족에게 알림 |

### 1.3 위치 공유

| ID | 구분 | 요구사항 | 우선순위 | 인수 기준 |

| REQ-011 | 실시간 지도 표시 | 가족 화면에서 부모님의 현재 위치를 카카오맵 기반 지도와 마커로 표시해야 한다 | Must | 지도 로드 후 3초 이내 마커 표시, 30초마다 마커 위치 자동 갱신 |

| REQ-013 | 현재 주소 표시 | 부모님의 현재 위치를 GPS 좌표가 아닌 도로명 주소로 표시해야 한다 | Should | 카카오맵 Geocoding API 활용, 주소 조회 실패 시 "위치 확인 중" 표시 |

## 2. 비기능 요구사항 (Non-Functional Requirements)

### 2.1 성능

| ID | 구분 | 요구사항 | 기준값 |
|----|------|----------|--------|
| NFR-001 | 응답 시간 | 페이지 최초 로드 시간 | 3초 이내 (LCP 기준, 4G 환경) |
| NFR-002 | SOS 지연 | SOS 버튼 확정 후 FCM 푸시 알림 수신까지 지연 시간 | 3초 이내 |
| NFR-003 | 위치 갱신 | 부모님 위치 전송 후 가족 화면 지도 마커 갱신까지 지연 시간 | 5초 이내 |

| NFR-005 | 가용성 | 서비스 Uptime | 99.5% 이상 (월 다운타임 약 3.6시간 이내) |
| NFR-006 | 동시 접속 | MVP 기준 동시 접속 사용자 수 | 최소 500명 동시 처리 |

| ID | 구분 | 요구사항 | 기준값 |
|----|------|----------|--------|

| NFR-010 | 위치정보법 준수 | 위치기반서비스사업자로 방통위에 신고하고, 위치정보법 준수 체계를 갖추어야 한다 | 서비스 출시 전 방통위 신고 완료 또는 신고 진행 중 |
| NFR-011 | 개인정보보호 | 개인정보보호법에 따른 개인정보처리방침 수립 및 고지 의무를 이행해야 한다 | 처리방침 페이지 필수 제공, 수집 항목·목적·보유 기간 명시 |

## 3. 우선순위 매트릭스 (MoSCoW)

| 우선순위 | 요구사항 ID | 요구사항 개요 | 비율 |
|----------|-------------|---------------|------|
| **Must** (필수) | REQ-001~010, REQ-014~017, REQ-022~024, REQ-028~029, NFR-001~012, NFR-013~018 | 회원가입·로그인, 위치정보 동의, 실시간 위치 공유, SOS 긴급 알림, 관리자 기본 기능, 보안·성능·접근성 기준 | 약 60% |
| **Should** (중요) | REQ-012~013, REQ-018~019, REQ-025, NFR-019~020 | 위치 이력 조회, 현재 주소 표시, 가족 채팅, 알림톡 발송, 운영 로깅·모니터링 | 약 20% |
| **Could** (선택) | REQ-020~021, REQ-026~027 | 지오펜싱, 배터리 절약 모드, 통계 대시보드 | 약 15% |
| **Won't** (이번 MVP 제외) | — | AI 비활동·이상행동 감지, 다국어 지원, 네이티브 앱 (iOS/Android) | 약 5% |
