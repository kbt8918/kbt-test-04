    Login --> |role: family + 그룹 미참여| FamilyGroup

    subgraph ParentView ["부모님 화면 (Parent View)"]
        subgraph Onboarding ["온보딩 플로우 (F-005)"]

## 2. 사용자 흐름 (User Flow)

### 2.1 부모님 최초 온보딩 플로우 (자녀 대리 설정)

```mermaid
flowchart LR
    Start([자녀가 부모님 단말 접속]) --> S1["1단계\n회원가입\n번호·역할(parent) 입력"]
    S1 --> S2["2단계\n그룹 초대 코드 입력\n(F-004 실행)"]
    S2 --> S3["3단계\n위치정보 동의 화면\n부모님 본인 직접 조작 필수"]
    S3 --> |필수 항목 동의 완료| S4["4단계\n온보딩 완료"]
    S3 --> |동의 거부| S4B["위치 공유 OFF 상태로 메인 진입\n동의 유도 배너 표시"]
    S4 --> ParentMain["부모님 메인 화면\n(SOS 버튼 중심)"]
    S4B --> ParentMain

    S2 --> |중간 이탈| Resume["localStorage 임시 저장\n재접속 시 이어서 진행"]
    Resume --> S2
```

### 2.2 가족 구성원 그룹 생성·참여 플로우

```mermaid
flowchart LR
    Start([가족 구성원 회원가입·로그인]) --> Choice{그룹 보유 여부}
    Choice --> |"그룹 없음\n(처음 서비스 시작)"| Create["그룹 생성 (F-003)\n그룹명 입력 → 6자리 초대 코드·QR 발급"]
    Choice --> |"초대 코드 보유"| Join["그룹 참여 (F-004)\n초대 코드 6자리 또는 QR 스캔"]
    Create --> FamilyMain["가족 메인 화면\n실시간 지도"]
    Join --> |코드 유효 + 정원 미달| FamilyMain
    Join --> |코드 오류| ErrCode["'초대 코드를 다시 확인해 주세요.' 안내"]
    Join --> |정원 초과| ErrFull["'그룹 인원이 가득 찼습니다.' 안내"]
    Join --> |이미 참여 중| ErrDup["'이미 참여한 그룹입니다.' 안내"]
    ErrCode --> Join
    ErrFull --> Choice
    ErrDup --> FamilyMain
```

### 2.3 SOS 긴급 알림 발송·수신 플로우

```mermaid
flowchart TD
    Parent["부모님: SOS 버튼 탭 (F-014)"] --> Countdown["2초 카운트다운 오버레이 표시 (F-015)\n전체 화면 반투명 레드 배경"]
    Countdown --> |취소 버튼 탭| Cancel["메인 화면 복귀"]
    Countdown --> |화면 이탈| Cancel
    Countdown --> |2초 경과 확정| Send["FCM SOS 알림 발송 (F-016)\n가족 그룹 전원 · 현재 위치 포함"]
    Send --> ParentToast["부모님: 'SOS를 보냈습니다.' 완료 토스트"]
    Send --> FamilyPush["가족: FCM 푸시 알림 수신"]
    FamilyPush --> FamilyModal["가족: SOS 수신 모달 (F-017)\n부모님 이름·위치 표시"]
    FamilyModal --> |전화 연결 버튼 탭| Call["네이티브 전화 앱 실행 (tel: scheme)"]
    FamilyModal --> |지도 보기 버튼 탭| Map["가족 메인 화면 지도 (F-011)"]

    Send --> |위치 없음| SendNoGPS["'위치 불명' 텍스트로 발송\n부모님에게 안내 메시지"]
    Send --> |FCM 실패| Retry["1회 재시도 후 실패 이력 저장\n부모님: '전송 실패' 안내"]
```

### 2.4 위치 공유·실시간 지도 갱신 플로우

```mermaid
flowchart LR
    Toggle["부모님: 위치 공유 ON 토글 (F-010)"] --> GPS["GPS 위치 수집\n30초 간격 (F-009)\nnavigator.geolocation.watchPosition"]
    GPS --> |정상 신호| Server["서버 저장\n(locations 테이블)"]
    GPS --> |신호 불량 accuracy > 100m| Skip["마지막 정상 위치 유지\n불량 데이터 저장 제외"]
    GPS --> |네트워크 단절| Queue["로컬 큐 최대 5건 임시 저장\n재연결 시 일괄 전송"]
    Server --> WS["WebSocket 실시간 전파 (/ws)"]
    WS --> FamilyMap["가족 메인 화면\n카카오맵 마커 갱신 (F-011)"]
    FamilyMap --> Geocode["역지오코딩 주소 표시 (F-013)\n카카오맵 coord2Address API"]
    FamilyMap --> |날짜 선택| History["위치 이력 조회\n이동 경로 폴리라인 표시 (F-012)"]

    Toggle2["부모님: 위치 공유 OFF 토글"] --> Stop["watchPosition 해제\n가족 화면 마커 제거"]
    Stop --> Notify["가족 전원 FCM 알림\n'위치 공유를 꺼두셨습니다.'"]
```

### 2.5 안전 구역 이탈 알림 플로우

```mermaid
flowchart LR
    Family["가족: 안전 구역 설정 (F-020)\n지도 탭 또는 주소 검색으로 중심점 지정\n반경 100m~5km 슬라이더"] --> Saved["geofences 테이블 저장\n카카오맵 원형 오버레이 시각화"]
    Saved --> Monitor["서버: 위치 데이터 수신마다\n Haversine 거리 계산 (F-021)"]
    Monitor --> |이탈 감지 후 30초 연속 확인| AlertOut["가족 전원 FCM\n'부모님이 [구역명]을 벗어났습니다.'"]
    Monitor --> |복귀 감지| AlertIn["가족 전원 FCM\n'부모님이 [구역명]으로 돌아오셨습니다.'"]
    Monitor --> |경계 근처 진동 flickering| Buffer["100m 버퍼 존 적용\n30초 연속 이탈 후 알림 발송"]
```

## 3. 화면-기능 매핑

| 화면명 | URL 경로 | View | 주요 기능 | 관련 기능 ID |
|--------|----------|------|-----------|-------------|
| 로그인 | `/login` | 공통 | 휴대폰 번호 로그인, 역할별 라우팅, 7일 자동 로그인 | F-002 |
| 회원가입 | `/register` | 공통 | 번호 입력, 역할 선택, 개인정보·이용약관 동의 | F-001, F-007 |
| 개인정보처리방침 | `/privacy` | 공통 | 처리방침 전문 열람 (정적 페이지) | F-028 |
| 온보딩 1단계 | `/onboarding/step1` | Parent | 부모님 번호·역할(parent) 입력 | F-005, F-001 |
| 온보딩 2단계 | `/onboarding/step2` | Parent | 그룹 초대 코드 입력 | F-005, F-004 |
| 온보딩 3단계 | `/onboarding/step3` | Parent | 위치정보 수집·이용 동의 (전체 화면 단독) | F-005, F-006 |
| 온보딩 4단계 | `/onboarding/step4` | Parent | 온보딩 완료, 메인 화면 진입 안내 | F-005 |
| 부모님 메인 | `/parent` | Parent | SOS 버튼, 위치 공유 ON/OFF 토글, 공유 상태 표시 | F-009, F-010, F-014 |
| SOS 카운트다운 오버레이 | `/parent` (오버레이) | Parent | 2초 카운트다운, SOS 확정 또는 취소 | F-015, F-016 |
| 부모님 설정 | `/parent/settings` | Parent | 위치정보 동의 철회, 배터리 절약 모드 전환 | F-008, F-027 |
| 그룹 생성 | `/family/group/create` | Family | 그룹명 입력, 초대 코드·QR 코드 발급 | F-003 |
| 그룹 참여 | `/family/group/join` | Family | 초대 코드 6자리 입력 또는 QR 스캔 | F-004 |
| 가족 메인 (지도) | `/family` | Family | 실시간 카카오맵, 부모님 위치 마커, 역지오코딩 주소 | F-011, F-013 |
| 위치 이력 | `/family/history` | Family | 날짜 선택, 이동 경로 폴리라인, 시간대별 위치 목록 | F-012 |
| 채팅방 | `/family/chat` | Family | WebSocket 실시간 채팅, 미접속자 FCM 푸시 알림 | F-018, F-019 |
| 안전 구역 설정 | `/family/geofence` | Family | 구역 생성·수정·삭제, 이탈·복귀 알림 활성화 | F-020, F-021 |
| SOS 수신 모달 | `/family` (모달) | Family | SOS 발신 부모님 이름·위치 표시, 전화 연결 | F-017 |
| 관리자 회원 목록 | `/admin/groups` | Admin | 가족 그룹 목록, 검색·정렬, 페이지네이션 (20건) | F-022 |
| 관리자 회원 상세 | `/admin/groups/:id` | Admin | 그룹 구성원 목록, 마지막 접속·위치 공유 상태 | F-023 |
| 관리자 SMS 발송 | `/admin/sms` | Admin | 대상 선택, SMS·LMS 자동 분류, 발송 이력 | F-024 |
| 관리자 알림톡 발송 | `/admin/alimtalk` | Admin | 템플릿 선택, 변수 입력, 발송 이력 (성공·실패 건수) | F-025 |

## 4. 화면별 진입 조건

| 화면명 | 진입 조건 | 미충족 시 처리 |
|--------|-----------|---------------|
| 부모님 메인 | 로그인 완료 + role=parent + 온보딩 완료 | 온보딩 1단계로 이동 |
| 온보딩 플로우 | 로그인 완료 + role=parent + 온보딩 미완료 | 미완료 단계로 자동 이동 |
| 부모님 설정 | 로그인 완료 + role=parent | 로그인 화면으로 이동 |
| 가족 메인 (지도) | 로그인 완료 + role=family + 그룹 참여 완료 | 그룹 생성·참여 화면으로 이동 |
| 위치 이력 | 로그인 완료 + role=family + 그룹 참여 완료 | 그룹 생성·참여 화면으로 이동 |
| 채팅방 | 로그인 완료 + role=family + 그룹 참여 완료 | 그룹 생성·참여 화면으로 이동 |
| 안전 구역 설정 | 로그인 완료 + role=family + 그룹 참여 완료 | 그룹 생성·참여 화면으로 이동 |
| 관리자 화면 전체 | 로그인 완료 + role=admin | 401 반환 + 로그인 화면으로 이동 |
| SOS 카운트다운 | 부모님 메인에서 SOS 버튼 탭 | 부모님 메인에서만 진입 가능 |
| SOS 수신 모달 | FCM SOS 알림 수신 또는 가족 메인 접속 시 미확인 SOS 존재 | — |
