## 1. Executive Summary

### 1.1 한 줄 요약

별도 앱 설치 없이 스마트폰 브라우저만으로 부모님 위치를 실시간 공유하고, 가족이 언제든 확인하며 즉시 소통할 수 있는 시니어 케어 통합 플랫폼

### 1.2 핵심 지표

| 항목 | 내용 |
|------|------|
| 서비스명 | 부모님 위치 확인 서비스 (브랜드명: 안심맵, AnsimMap) |
| 목표 시장 | 한국 시니어 케어·가족 안전 위치 추적 서비스 시장 |
| 타겟 사용자 | 고령 부모님을 둔 3040 자녀 / 디지털 기기에 익숙하지 않은 6070 시니어 |
| 핵심 가치 | 안심(실시간 위치 파악) + 간편성(시니어 초간편 UX) + 연결성(위치·SOS·채팅 원스톱) |
| 수익 모델 | B2C Freemium 구독 + B2G 지자체·복지관 계약 이중 구조 |
| 목표 기간 | 2026-05-18 ~ 2026-06-21 (MVP 5주) |

## 4. 서비스 전략

### 4.1 핵심 가치 제안 (Value Proposition)

> 부모님은 버튼 하나로 안전을 알리고, 가족은 지도 한 장으로 안심한다.

고령의 부모님을 둔 가족이 겪는 핵심 문제 — 실시간 위치 파악 불가, 긴급 상황 인지 지연, 반복 안부 전화 부담 — 를 별도 앱 설치 없이, 스마트폰 브라우저 하나로 해결한다. 시니어를 위한 초간편 UX(SOS 버튼 중심, 글씨 18px+, 버튼 48×48px+)와 가족을 위한 실시간 카카오맵 지도·채팅·전화 통합을 단일 서비스에서 제공한다.

### 4.2 서비스 범위

| 구분 | 기능/서비스 | 우선순위 | 설명 |
|------|------------|----------|------|
| MVP | 휴대폰 번호 및 구글 소셜 회원가입·로그인 | P0 | 휴대폰 번호 또는 구글 계정을 이용한 간편 로그인/가입 |
| MVP | 스마트 랜딩 및 예외 처리 | P0 | 부모님 존재 여부에 따른 랜딩 화면 자동 분기 및 중복/오류 배너 알림 |

### 4.3 차별화 전략
