## 1. 디자인 핵심 원칙 (Design Principles)

안심맵의 디자인은 디지털 기기 사용이 서툰 시니어(부모님)와 부모님의 안전을 모니터링하는 자녀(가족 구성원) 모두에게 최적의 사용성을 제공하도록 설계되었습니다.

> [!IMPORTANT]
> ### 👵 1. 시니어 우선 설계 (Senior-First UX)
> *   **가독성 중심:** 글씨 크기는 최소 **18px 이상**을 기본 스케일로 지정하며, 작은 폰트(12px 이하)의 사용을 배제합니다.
> *   **오조작 방지:** 주요 대화형 요소 및 버튼은 최소 **48px × 48px** 이상의 터치 영역(Touch Target)을 확보합니다.
> *   **플레이스홀더 배제:** 중요 필드 입력 칸은 플레이스홀더 텍스트 대신 명확하고 고정된 레이블을 상단에 고정 표시합니다.

> [!TIP]
> ### 💚 2. 따뜻함과 신뢰를 주는 안심 디자인 (Safety & Trust)
> *   **차분한 색채:** 강렬하고 불안감을 줄 수 있는 임시 원색을 피하고, 심리적 안정감을 주는 **포레스트 그린(Forest Green)**과 밝고 생기 있는 **민트 그린(Mint Green)**을 핵심 브랜드 컬러로 삼습니다.
> *   **긴급성 제어:** 비상 호출(SOS) 영역에는 뚜렷하게 식별 가능한 채도 높은 **크림슨 레드(Crimson Red)**와 맥박처럼 뛰는 마이크로 애니메이션을 제공하여 시각적 직관성을 최대화합니다.

## 2. 글로벌 레이아웃 & 그리드 시스템

### 2.1 화면 규격 및 영역 정의 (Breakpoints)

*   **모바일 웹 (기본 뷰포트: 375px ~ 768px):** 부모님(Parent) 및 자녀(Family) 메인 모바일 웹 뷰포트. 세로 단일 스크롤 최적화.
*   **PC 웹 (1920px 해상도, 최대 너비 1440px 고정):** 관리자(Admin) 통합 관제 시스템 전용.

```text
[ 모바일 레이아웃: 375px ]             [ PC 어드민 레이아웃: 1440px ]
+-------------------------+          +-----------------------------------------------------+
|      Header (56px)      |          |                   Header (64px)                     |
+-------------------------+          +-----------------+-----------------------------------+
|                         |          |                 |                                   |
|                         |          |                 |                                   |
|      Main Content       |          |  Sidebar        |           Main 관제 영역          |
|      (Fluid / 100vh)    |          |  (260px)        |           (Fluid grid)            |
|                         |          |                 |                                   |
|                         |          |                 |                                   |
+-------------------------+          +-----------------+-----------------------------------+
|  Bottom Nav Bar (60px)  |          
+-------------------------+          
```

### 2.2 스페이싱 토큰 (Spacing Scale - 8px Base Grid)

모든 컴포넌트 간격, 패딩, 마진은 **8px 그리드 스케일**을 기반으로 구조화하여 완벽한 시각적 일관성을 확보합니다.

| 토큰명 | 크기 | 용도 및 사용 예시 |
|:---|:---|:---|
| **sp-xs** | `4px` (0.25rem) | 텍스트 내 배지 부착, 아이콘-텍스트 내부 결합 간격 |
| **sp-sm** | `8px` (0.5rem) | 리스트 항목 내 텍스트 행간 간격, 인접 소형 요소 |
| **sp-md** | `16px` (1rem) | 컴포넌트 내부 기본 패딩, 일반 인풋 필드 간격 |
| **sp-lg** | `24px` (1.5rem) | 카드 컴포넌트 패딩, 섹션 내 하위 요소 격차 |
| **sp-xl** | `32px` (2rem) | 본문 콘텐츠 영역 외부 여백, 주요 섹션 간 간격 |
| **sp-2xl**| `48px` (3rem) | 대형 버튼 높이 확보, 온보딩 단계 간 여백 |
| **sp-3xl**| `64px` (4rem) | 랜딩 페이지 히어로 여백, 빈 상태(Empty State) 정중앙 공간 |

## 3. 컬러 시스템 (Color System)

안심맵의 색채 계획은 HSL(Hue-Saturation-Lightness) 값을 조절하여 부드럽고 고급스러우며 대비가 명확하여 저시력자 및 시니어도 편리하게 읽을 수 있도록 세심히 조색되었습니다.

### 3.1 브랜드 메인 컬러 (Primary Color Set)

```text
■ Forest Green (Primary)     ■ Mint Green (Accent)       ■ Primary Light (Background)
#2E7D32 / HSL(122, 45%, 34%)  #4CAF50 / HSL(122, 39%, 49%)  #E8F5E9 / HSL(125, 39%, 94%)
```

*   **Primary (Forest Green - `#2E7D32`):** 주요 안심 브랜드 아이덴티티, 액션 버튼, 활성 탭 하이라이트.
*   **Primary Dark (Deep Forest - `#1B5E20`):** 타이틀 텍스트, 헤더 바 배경, 어드민 상단 내비게이션.
*   **Primary Light (Soft Mint - `#E8F5E9`):** 긍정 상태 카드 배경, 활성 입력 필드 틴트배경.

### 3.2 뉴트럴 톤 팔레트 (Neutral Gray Palette)

텍스트 계층 구조 및 외곽선 구분을 위한 세련된 모노톤 팔레트입니다.

| 컬러 토큰 | Hex 값 | 사용 기준 (Usage Guide) |
|:---|:---|:---|
| **Gray 50** | `#FAFAFA` | 서비스 메인 슬라이드 백그라운드 색상 |
| **Gray 100** | `#F5F5F5` | 일반 카드 컴포넌트 기본 배경 |
| **Gray 200** | `#EEEEEE` | 인라인 컴포넌트 경계선, 테두리, 구분선 |
| **Gray 300** | `#E0E0E0` | 비활성 입력창 배경, 폼 비활성화 버튼 |
| **Gray 400** | `#BDBDBD` | 입력 폼 가이드 캡션, 플레이스홀더 텍스트 |
| **Gray 500** | `#9E9E9E` | 부모님 단말기 위치 갱신 정보(오래됨) 회색 배지 |
| **Gray 600** | `#757575` | 일반 보조 설명 텍스트, 비활성 탭 아이콘 |
| **Gray 700** | `#616161` | 일반 대화창 자녀 말풍선 내부 보조 텍스트 |
| **Gray 800** | `#424242` | 일반 본문 콘텐츠 텍스트, 다이어로그 내 설명 |
| **Gray 900** | `#212121` | 메인 헤드라인 제목, 타이틀, 강조 본문 서체 |

### 3.3 시맨틱 및 맵 그래픽 컬러 (Semantic & Graphic Overlay)

*   **Danger (Crimson Red - `#D32F2F`):** SOS 긴급 버튼, SOS 카운트다운 배경, 안전 구역 이탈 알람 텍스트 및 경보 아이콘.
*   **Danger Light (Soft Red - `#FFEBEE`):** SOS 수신 팝업 다이얼로그 배경색.
*   **Success (Deep Blue/Teal - `#1976D2`):** 안전 구역 진입 완료 배지, 정상 확인 상태 배지.
*   **Map Safe Zone Circle (`rgba(46, 125, 50, 0.15)`):** 실시간 카카오맵 상에 드로잉되는 안전 구역 반투명 민트-녹색 오버레이 서클.
*   **Map Breached Zone Circle (`rgba(211, 47, 47, 0.15)`):** 이탈 감지 순간 적색으로 깜빡이는 위험 지오펜스 오버레이 서클.

## 4. 타이포그래피 (Typography)

시니어층의 사용성을 해치지 않기 위해 기본 타입 페이스를 표준 Pretendard로 지정하며, 글꼴 크기는 대시보드를 제외하면 13px 이하를 지양합니다.

### 4.1 타입 페이스 지정 (Font Family)

*   **UI 본문 및 제목:** `Pretendard`, `-apple-system`, `BlinkMacSystemFont`, `Apple SD Gothic Neo`, `Malgun Gothic`, `sans-serif`
*   **정밀 주소 및 소켓 좌표 데이터:** `Monaco`, `Consolas`, `monospace` (숫자 정렬 및 오독 방지용 고정폭 지정)

### 4.2 타이포그래피 스케일 정의 (Type Scale)

```text
[ Pretendard Typography Hierarchy ]
display-lg (36px / Bold)  ===>  🚨 2초 카운트다운 대형 숫자
heading-1 (24px / Bold)   ===>  모바일 주요 헤더 타이틀, 대형 CTA 버튼
heading-2 (20px / Bold)   ===>  시니어용 메인 가이드 텍스트 (18px~20px)
heading-3 (16px / Med)    ===>  자녀용 피드 및 리스트 카드 제목
body-lg (16px / Reg)      ===>  자녀용 본문 대화 텍스트
body-sm (13px / Reg)      ===>  타임스탬프 및 서브 정보
caption (12px / SemiB)    ===>  상태 배지용 텍스트
```

| 토큰명 | 크기 (font-size) | 행간 (line-height) | 자간 (letter-spacing) | 두께 (font-weight) | 실적용 예시 |
|:---|:---|:---|:---|:---|:---|
| **display-lg**| `36px` (2.25rem) | `44px` (1.2) | `-0.02em` | **Bold (700)** | SOS 카운트다운 대형 숫자 |
| **heading-1** | `24px` (1.5rem) | `32px` (1.33) | `-0.01em` | **Bold (700)** | 회원가입/로그인 헤더, 대형 버튼 |
| **heading-2** | `20px` (1.25rem) | `28px` (1.4) | `0em` | **SemiBold (600)** | **부모님 전용 UI 메인 가이드 텍스트** |
| **heading-3** | `16px` (1rem) | `24px` (1.5) | `0em` | **SemiBold (600)** | 자녀용 카테고리 탭, 카드 타이틀 |
| **body-lg**   | `16px` (1rem) | `24px` (1.5) | `0em` | Regular (400) | 자녀용 본문 텍스트, 채팅 말풍선 |
| **body-sm**   | `13px` (0.81rem) | `18px` (1.38) | `0.01em` | Regular (400) | 타임스탬프, 역지오코딩 세부 텍스트 |
| **caption**   | `12px` (0.75rem) | `16px` (1.33) | `0.02em` | **SemiBold (600)** | 상태 배지 내부 약어, 배터리 퍼센트 |
| **overline**  | `11px` (0.68rem) | `14px` (1.27) | `0.08em` | **Bold (700)** | 카드 최상단 소형 태그 카테고리 |

## 5. 아이콘 가이드라인 (Icon System)

안심맵의 모든 UI 아이콘은 기하학적인 직관성과 따뜻함을 제공하기 위해 외곽선 스타일(Outlined Stroke)을 기본으로 설계되었습니다.

### 5.1 아이콘 스타일 원칙
*   **스타일 패밀리:** `Lucide Icons` (또는 `Heroicons Outline`) 적용 일치 권장.
*   **선 두께 (Stroke Width):** 기본 모바일 뷰 `1.5px` 적용. 단, 모바일 내부 16px 이하 소형 아이콘의 가독성을 위해 소형 뷰포트에서는 `2px` 두께로 예외 적용.
*   **선 끝 형태:** 부드럽고 신뢰감을 주도록 둥근 코너 값(`stroke-linejoin: round; stroke-linecap: round;`) 지정.

### 5.2 아이콘 크기 규격

*   `16px` (Micro): 인라인 텍스트 내부 배터리 잔량 표시, 타임라인 리스트 체크 핀.
*   `20px` (Small): 모바일 하단 내비게이션 바 메뉴, 인풋 필드 내 돋보기 아이콘.
*   `24px` (Medium): 일반 카드 컴포넌트 헤더, 설정 세부 행 목록 구분자.
*   `32px` (Large): "그룹이 없습니다" 등 빈 상태 일러스트용 아이콘.
*   `48px` (X-Large): SOS 성공 확인, 온보딩 가입 성공 체크.

## 6. 핵심 UI 구성 요소 (Core UI Components)

### 6.1 버튼 디자인 스펙 (Buttons)

```text
[ 모바일 표준 버튼 스펙 ]
+---------------------------------------+
|                확 인                  |  H: 48px / HSL(122, 45%, 34%)
+---------------------------------------+  Rounded-radius: 8px / Text: heading-2 Bold

[ 부모님 SOS 전용 초대형 원형 버튼 ]
    .   ,
  '       '
 |   🚨   |   W & H: 120px 원형 (bg-red-600)
  .  SOS .    Shadow: 0px 8px 24px rgba(211, 47, 47, 0.4)
    '   '
```

*   **Primary Button:**
    *   **스타일:** `bg-forest text-white hover:bg-forest-dark`
    *   **크기:** 모바일 높이 **48px 고정** (PC 어드민 40px), 둥글기 테두리 `8px (rounded-lg)` 적용.
    *   **폰트:** `heading-2 (20px) / Bold` (부모님 뷰), `heading-3 (16px) / SemiBold` (자녀 뷰).
*   **SOS Large Button (부모님 메인):**
    *   **스타일:** 붉은색(`bg-red-600`) 원형 아이콘. 지름 **120px 이상** 크기 보장.
    *   **그림자:** `box-shadow: 0 8px 24px rgba(211, 47, 47, 0.4);`로 시각적 입체감 부여.
    *   **효과:** 무한 루프로 2초마다 은은하게 번지는 Ripple(파동) 애니메이션 탑재 필수.

### 6.2 입력 필드 스펙 (Inputs)

*   **크기:** 세로 높이 **48px** 확보 (터치 최적화).
*   **디폴트 보더:** `border: 1.5px solid` `Gray 200` (`#EEEEEE`).
*   **포커스(Focus) 상태:** `border-color: Forest Green` (`#2E7D32`), 배경색 `Soft Mint` (`#E8F5E9`) 틴트 전환.
*   **에러(Error) 상태:** `border-color: Crimson Red` (`#D32F2F`), 인라인 가이드 텍스트 `Danger Red`로 노출.

### 6.3 카드 & 대화 상자 (Cards & Modals)

*   **카드 컴포넌트:** 테두리 `rounded-2xl` (`16px`), 그림자 `shadow-sm` (`box-shadow: 0 2px 8px rgba(0, 0, 0, 0.05);`).
*   **다이얼로그 모달:**
    *   **부모님용:** 화면 중앙이 아닌 **바텀시트(BottomSheet) 형태**로 하단에서 슬라이드 업 하도록 설계하여 모바일 한 손 조작성 강화.
    *   **자녀용 SOS 팝업:** 긴급 상황을 상징하도록 최상층 레이어(`z-index: 9999`)에 위치하며, 화면 전체에 딤 처리(`rgba(0, 0, 0, 0.6)`) 및 모달 테두리에 2px 두께의 레드 외곽선 추가 적용.

## 7. 인터랙션 및 애니메이션 토큰

안심맵의 인터랙션은 시니어층의 어지러움이나 인지 지연을 유발하지 않도록 과도한 3D 효과를 배제하고 자연스럽고 은은한 마이크로 애니메이션으로 처리됩니다.

### 7.1 화면 트랜지션 (Transitions)

*   **기본 컴포넌트 호버/액티브:**
    *   `transition: all 0.2s cubic-bezier(0.4, 0, 0.2, 1);`
    *   버튼 터치 시 `scale: 0.96`으로 약간 축소되는 햅틱 반응성 모사 제공.
*   **페이지 전환:**
    *   모바일 웹 내 화면 슬라이드: 좌우 슬라이드(`slide-in-from-right`, `duration-300`) 애니메이션을 부여해 모바일 하이브리드 앱과 유사한 네이티브 경험 선사.

### 7.2 마이크로 애니메이션 특수 토큰

*   **SOS Pulse Animation (SOS 버튼 활성 상태):**
    ```css
    @keyframes sos-pulse {
      0% {
        box-shadow: 0 0 0 0 rgba(211, 47, 47, 0.4);
      }
      70% {
        box-shadow: 0 0 0 20px rgba(211, 47, 47, 0);
      }
      100% {
        box-shadow: 0 0 0 0 rgba(211, 47, 47, 0);
      }
    }
    .animate-sos-pulse {
      animation: sos-pulse 2s infinite;
    }
    ```
