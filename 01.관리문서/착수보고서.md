## 1. 프로젝트 개요

### 1.1 배경 및 필요성

고령의 부모님을 둔 가족은 치매·낙상·독거 어르신 등 안전 사각지대에 놓인 부모님의 현재 위치를 실시간으로 파악하기 어렵다. 기존 위치 추적 서비스는 전용 기기 구매 또는 앱 설치를 요구해 디지털 기기에 익숙하지 않은 시니어에게 진입 장벽이 높다.

본 프로젝트는 **별도 앱 설치 없이 모바일 웹 브라우저만으로** 부모님이 위치를 공유하고, 가족 구성원이 실시간으로 위치를 확인할 수 있는 반응형 웹 플랫폼을 구축한다.

### 1.2 프로젝트 기본 정보

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 부모님 위치 확인 서비스 |
| 서비스 유형 | 시니어 케어 · 가족 안전 위치 추적 반응형 웹 플랫폼 |
| 플랫폼 | 반응형 웹 (모바일 360px~ / 태블릿 768px~ / PC 1280px~) |
| 프로젝트 기간 | 2026-05-18 ~ 2026-06-21 (5주) |
| 기술 스택 | Next.js + Tailwind CSS / Next.js API Routes / PostgreSQL (Supabase) / Vercel |

## 2. 프로젝트 목표

### 2.1 단계별 목표

| 구분 | 목표 | 측정 지표 |
|------|------|-----------|
| 단기 — Phase 1 MVP | 부모님 실시간 위치 공유 + SOS 긴급 알림 기능 구현 | 위치 업데이트 지연 5초 이내, SOS 알림 전송 3초 이내 |
| 중기 — Phase 2 | 가족 채팅·위치 이력·관리자 카카오톡 발송 기능 추가 | 7일 이력 조회, 채팅 메시지 전송 성공률 99% 이상 |
| 장기 — Phase 3 | 통계 대시보드·배터리 최적화·다수 부모님 지원 | 서비스 Uptime 99.5% 이상 |

### 2.2 핵심 가치

| 가치 | 설명 |
|------|------|
| 안심 | 가족이 언제든 부모님의 현재 위치를 실시간으로 파악 가능 |
| 간편성 | 부모님이 스마트폰 단순 조작만으로 위치 공유 및 긴급 신호 전송 |
| 연결성 | 가족 간 채팅·전화 기능으로 즉각적인 소통 지원 |

## 3. 서비스 사용자 및 핵심 기능

### 3.1 사용자 역할

| 역할 | 설명 | 주요 화면 |
|------|------|-----------|
| 부모님 (위치 제공자) | 스마트폰 브라우저에서 위치를 공유하는 시니어 사용자 | 부모님 화면 (Parent View) |
| 자녀/가족 (위치 확인자) | 웹에서 부모님 위치를 조회하는 가족 구성원 | 가족 화면 (Family View) |
| 운영 관리자 | 회원 관리·SMS·카카오톡 발송을 담당하는 운영사 담당자 | 관리자 화면 (Admin View) |
