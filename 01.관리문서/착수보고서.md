## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 부모님 위치 확인 서비스 |
| 서비스 유형 | 시니어 케어 · 가족 안전 위치 추적 반응형 웹 플랫폼 |
| 서비스 목적 | 고령의 부모님 위치를 가족이 실시간으로 확인하고, 긴급 상황 시 신속하게 대응할 수 있도록 지원 |
| 플랫폼 | 반응형 웹 (모바일 360px~ / 태블릿 768px~ / PC 1280px~) |
| 기술 스택 | Next.js + Tailwind CSS / Express.js / PostgreSQL / Vercel (V0.42 표준) |
| 프로젝트 기간 | 2026-05-18 ~ 2026-06-21 (5주) |

### 서비스 개요

디지털 기기에 익숙하지 않은 시니어(부모님)도 단순 조작만으로 긴급 알림을 발송할 수 있으며, 가족 구성원은 실시간 위치 지도·채팅·전화 연결을 하나의 서비스에서 이용할 수 있다. 운영사 관리자는 회원(가족 그룹) 관리와 SMS·카카오톡 알림 발송을 전담 어드민 화면에서 수행한다.

## 2. 프로젝트 목표

| 구분 | 목표 | 측정 지표 |
|------|------|-----------|
| 단기 (Phase 1 MVP) | 부모님 실시간 위치 공유 + SOS 긴급 알림 기능 구현 | 위치 업데이트 지연 5초 이내, 알림 전송 3초 이내 |
| 중기 (Phase 2) | 가족 채팅·위치 이력·관리자 카카오톡 발송 기능 추가 | 7일 이력 조회, 채팅 메시지 전송 성공률 99% 이상 |
| 장기 (Phase 3) | 통계 대시보드·배터리 최적화·다수 부모님 지원 | 서비스 Uptime 99.5% 이상, 배터리 소모 최소화 |

### 핵심 가치

| 가치 | 설명 |
|------|------|
| 안심 | 가족이 언제든 부모님의 현재 위치를 실시간으로 파악 가능 |
| 간편성 | 부모님이 스마트폰 단순 조작만으로 긴급 신호 전송 가능 |
| 연결성 | 가족 간 채팅·전화 기능으로 즉각적인 소통 지원 |

## 3. 프로젝트 범위

### 3.1 범위 내 (In Scope)

| 영역 | 기능 | 우선순위 |
|------|------|---------|
| 공통 | 휴대폰 번호 기반 회원가입/로그인, 가족 그룹 코드 연결, 위치정보 동의 처리 | P0 |
| 부모님 화면 | 실시간 GPS 위치 공유, SOS 긴급 알림 버튼, 오조작 방지(2초 카운트다운) | P0 |
| 부모님 화면 | 배터리 절약 모드, 상태 메시지 | P2 |
| 가족 화면 | 카카오맵 기반 실시간 위치 지도, 바로 전화 연결 버튼, FCM 긴급 알림 수신 | P0 |
| 가족 화면 | 위치 이력 조회(최대 7일), 가족 채팅방, 마지막 확인 시각 표시 | P1 |
| 가족 화면 | 다수 부모님 지원 | P2 |
| 관리자 화면 | 회원(가족 그룹) 목록·상세 조회, SMS 발송 | P0 |
| 관리자 화면 | 회원 검색·필터, 카카오 알림톡 발송, 발송 이력 조회, 계정 정지/복구 | P1 |
| 관리자 화면 | 서비스 이용 통계 대시보드 | P2 |

### 3.2 범위 외 (Out of Scope)

| 항목 | 사유 |
|------|------|
| 네이티브 앱(iOS/Android) 개발 | 반응형 웹(PWA)으로 대체, 1차 출시 후 검토 |
| 결제·구독 요금제 기능 | 과금 모델 미확정, 별도 프로젝트로 진행 |
| 타 서비스(카카오맵 외 지도 API) 병행 지원 | 카카오맵 API 단일 사용으로 범위 확정 |
| 관리자의 실시간 위치 직접 접근 | 개인정보보호법 준수를 위해 관리자에게 개인 위치 데이터 비공개 |
