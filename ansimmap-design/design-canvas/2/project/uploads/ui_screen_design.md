### SCR-001. 로그인/회원가입

| 항목 | 내용 |
|------|------|
| URL | `/login` (회원가입은 동일 페이지 내 탭 전환 `/register`) |

| 관련 API | `POST /api/auth/login`, `POST /api/auth/register` |

#### 🔄 예외 처리 및 상태
* **동의 미체크 상태:** 회원가입 모드에서 약관 동의 체크박스가 해제되어 있으면 '시작하기' 버튼이 연한 회색(`bg-gray-300`)으로 비활성화되며 클릭이 차단됩니다.
* **가입되지 않은 번호 로그인 시도:** "등록되지 않은 번호입니다. 회원가입 탭을 클릭하여 가입해 주세요." 경고 토스트 팝업이 노출됩니다.

#### 🧩 UI 컴포넌트

| 컴포넌트명 | 타입 | 설명 | 인터랙션 |
|:---|:---|:---|:---|
| **초대 코드 입력창** | TextInput | 가족 구성원 그룹 매핑을 위한 6자리 코드 입력 | 대문자 영문 및 숫자만 입력 가능, 입력 완료 시 버튼 활성화 |
| **위치정보 동의 토글** | Checkbox | 위치정보법 제15조 준수를 위한 법정 필수 동의 | 필수 약관 동의 체크 시에만 '위치 공유 시작하기' 버튼이 활성화됨 |
| **대리 조작 방지 경고** | Callout | 법적 요건 강화를 위한 주의 문구 영역 | 주황색 경고 테두리와 함께 부모님 직접 조작을 유도하는 시각적 강조 |
| **시작하기 버튼** | Button | 최종 온보딩 승인 및 Geolocation API 권한 트리거 | 탭 시 브라우저 `navigator.geolocation` 권한 요청 모달 팝업 트리거 |

#### 🔄 예외 처리 및 상태
* **중간 이탈 처리:** 사용자가 온보딩 입력 도중 이탈할 경우 현재 진행 단계를 `localStorage`에 자동 저장하고, 재진입 시 끊긴 단계부터 즉시 재개합니다.
* **브라우저 위치 권한 거부:** Geolocation 권한 요청에서 '거부'를 누른 경우, 온보딩은 완료되지만 부모님 메인 화면 최상단에 "위치 권한 꺼짐: 활성화 방법 보기" 가이드 배너를 표시합니다.

|   +-------------------------------+   | |                  (( 2 ))              |
|   |  현재 부모님 위치 전송 중...  |   | |                                       |

#### 🧩 UI 컴포넌트

| 컴포넌트명 | 타입 | 설명 | 인터랙션 |
|:---|:---|:---|:---|
| **SOS 긴급 호출 버튼** | LargeButton | 화면 전체의 50% 이상 면적을 차지하는 레드 구형 버튼 | 1회 터치 시 SOS 카운트다운 전체화면 오버레이 즉시 표시 |
| **위치 공유 토글** | ToggleSwitch | 실시간 GPS 수집 및 전송 스케줄러 일시 중단/재개 | 토글 변경 시 즉시 수집 활성/비활성 처리 및 가족에게 알림 푸시 발송 |
| **SOS 카운트다운 타이머** | Text | 오조작을 방지하기 위한 2초 지연 시각화 | 2초에서 시작해 0초가 될 때까지 크기가 맥박처럼 고동치는 애니메이션 |
| **발송 취소 버튼** | Button | 오작동으로 인한 SOS 트리거를 즉시 철회 | 카운트다운 중에 탭하면 오버레이가 즉시 닫히고 평시 메인으로 복귀 |
| **설정 바로가기** | IconButton | 동의 철회 및 배터리 절약 모드로 진입하는 톱니바퀴 아이콘 | 클릭 시 설정 화면(`/parent/settings`)으로 이동 |

#### 🔄 예외 처리 및 상태
* **백그라운드 차단 방지 배너 (iOS):** 부모님이 Safari 브라우저에서 화면을 닫거나 홈 화면으로 나갈 경우, 포그라운드 연결을 유지해 달라는 알림 팝업 및 홈 화면 설치(PWA) 권장 안내를 시각적으로 지속 노출합니다.
* **SOS 발생 시 화면 변화:** 카운트다운이 끝나고 SOS가 최종 발송되면, 전체 화면 배경이 어두워지며 "가족에게 긴급 알림을 보냈습니다." 알림 토스트와 함께 네이티브 사이렌 진동 효과를 3초간 가동합니다.
